			## 컨테이너 태그

-HTML의 컨테이너(Container) 태그는 컨텐츠나 레이아웃에 아무런 형향을 주지않고 
  HTML 요소들을 그룹화하거나 포함하기 위해 사용되는 태그이다.

-컨테이너 태그는 레이아웃을 구조화하거나 스타일을 적용하기 위해 매우 중요한 역할을 한다.

-컨텐츠나 레이아웃에 아무런 형향을 주지않고 다른 요소 여럿을 묶어 
 관리하기 편하게 만드는 역할을 하는 태그를 컨테이너 태그라고 한다.
-컨텐츠를 구분하거나 공통적인 스타일을 적용하고자 할때 개발자는 컨테이너를 사용한다.



	# 블럭 레벨 요소 (Block-level Element) 

-자기가 속한 영역의 너비를 모두 차지하여 블록을 형성한다.

-<div></div>	: Division의 약자이며, 문서의 구역 또는 블록을 정의하는 데 사용
-블록 레벨 요소(Block-level Element)로, 다른 요소들과 줄바꿈이 자동으로 발생.
-기본적으로 어떤 스타일도 적용되지 않은 중립적인 태그.

-태그의 주요 용도
 # <div>는 웹 페이지의 다양한 레이아웃(헤더, 콘텐츠, 사이드바 등)을 정의할 때 사용
 # CSS와 함께 사용하여 특정 영역에 스타일을 적용하거나 배치.




	# 인라인(inline Element) 요소

-자기에게 필요한 만큼의 공간만 차지한다.
-<span>	</span> :  태그는 HTML에서 인라인(inline) 요소로 사용되며, 텍스트나 다른 인라인 요소를 
 		   특정 부분만 그룹화하거나 스타일을 적용할 때 사용
-인라인(inline) 요소로, 텍스트와 함께 사용되거나 다른 인라인 요소와 한 줄에 배치.
기본적으로 시각적인 변화는 없으며, 스타일링이나 DOM 조작을 위해 사용.

		## 전역 속성(Global attributes)

-전역속성은 모든 HTML 태그에서 공통을 사용하는 태그이다.
-속성이란 태그의 부가적인 기능을 정의하는 것으로 선택사항이다.
-속성은 시작 태그의 내부에 정의한다.
-속성의 개수에는 제한이 없다.

	<태그명 속성명="속성값"  속성명="속성값">컨텐츠</태그명>

-대표적인 전역 속성
 # id	: 요소에 고유한 이름을 부여하는 식별자 역할 속성 (html에서 id값은 유일해야 한다.)
 # class	: 요소를 그룹별로 묶을수 있는 식별자 역할 속성 (class는 중복 가능)
 # style	: 요소에 적용할 CSS 스타일을 선언하는 속성
 # title	: 요소의 추가 정보를 제공하는 텍스트 속성 (사용자에게 툴팁을 제공)
