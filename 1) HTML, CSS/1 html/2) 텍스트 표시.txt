			## 텍스트 표시

	# 제목(headline)

# <h> 태그는 제목을 요소하는 태그이다.
# <h> 태그는 숫자와 함께 사용되며 숫자 1이 가장 크고 숫자 6이 가장 작다.

<body>
    <h1>안녕하세요 : h1 태그</h1>
    <h2>안녕하세요 : h2 태그</h2>
    <h3>안녕하세요 : h3 태그</h3>
</body>







	# 문단(paragraph)

# <p> 태그는 문단의 요소를 나타내는 태그로써 가장 많이 사용되는 텍스트 태그이다.
# 하나의 p 태그는 하나의 문단을 표현한다.
# 문단과 문단 사이에는 공백이 있다.

-HTML은 공백도 한번만 인정한다.
