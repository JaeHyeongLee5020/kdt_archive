		## 자바스크립트의 if 문 동작 방식

-if 문은 조건식 안의 값을 불리언(Boolean)으로 변환하여 평가한다.
  즉, 자바스크립트는 조건식의 값이 참(true)인지 거짓(false)인지를 판단하기 위해 내부적으로 다음 과정을 수행합니다:

1) 조건식의 값이 불리언 값이 아니면 자바스크립트는 자동으로 해당 값을 불리언 타입으로 변환
2) 변환된 결과가 true라면 블록 내부의 코드가 실행되고, false라면 실행되지 않는다.




	# Truthy와 Falsy 값
-자바스크립트에서 조건식으로 사용될 수 있는 값은 Truthy(참처럼 평가되는 값)와 Falsy(거짓처럼 평가되는 값)로 나뉜다.

-Falsy 값 (false처럼 평가됨)
-아래 값들은 모두 false로 평가됩니다:
 # false (불리언 값)
 # 0 (숫자)
 # -0 (숫자)
 # "" 또는 '' (빈 문자열)
 # null
 # undefined
 # NaN (Not-a-Number)

-Truthy 값 (true처럼 평가됨)
-Falsy 값이 아닌 모든 값은 true로 평가된다.
 # true (불리언 값)
 # 숫자: 1, -1, 42 등 (0이 아닌 모든 숫자)
 # 문자열: "Hello", 'World' 등 (빈 문자열 제외)
 # 객체: {}, [] 등
 # 함수: function () {}
